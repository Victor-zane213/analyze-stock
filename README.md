# Financial Assistant — A股/美股股票分析助手

一个基于 Claude Code 的股票分析 Skill，支持 **A 股（沪深）** 和 **美股** 的单只/批量分析，覆盖基本面、技术面、产业链、估值模型、牛散持仓、板块排名、涨停复盘等完整分析链路。

## 快速开始

### 1. 安装依赖

```bash
pip install -r requirements.txt
```

依赖包：`requests`, `pandas`, `pyyaml`, `openai`, `yfinance`, `akshare`

### 2. 配置 LLM

编辑 `config.yaml` 设置大模型 API（用于产业链分析和故事打分）：

```yaml
llm:
  provider: deepseek        # 默认服务商
  providers:
    deepseek:
      base_url: "https://api.deepseek.com"
      api_key: "你的API Key"
      model: deepseek-v4-pro
    dashscope:
      base_url: "https://dashscope.aliyuncs.com/compatible-mode/v1"
      api_key: "你的API Key"
      model: qwen-plus
```

### 3. 在 Claude Code 中加载 Skill

将本目录放入 `~/.claude/skills/` 下，Claude Code 会自动识别。之后在对话中使用触发词即可调用对应分析模式。

## 十种分析模式

### 模式 A：单只 A 股详细分析

对单只 A 股进行完整八步深度分析：基本面 → 盈利 → 技术指标 → 产业链 → 关联公司 → 综合判断 → 巴菲特框架 → 估值模型。

| 触发方式 | 示例 |
|---------|------|
| `分析A股 {股票名}` | `分析A股 贵州茅台` |
| `分析 {股票名}` | `分析 药明康德` |
| `{股票名} 怎么样` | `寒武纪 怎么样` |

**数据源**：akshare（财务）+ 腾讯行情（K线/均线）+ LLM（产业链）+ WebSearch（回退）

---

### 模式 B：多只 A 股批量对比

批量分析多只 A 股，输出汇总对比表 + 推荐排序。

| 触发方式 | 示例 |
|---------|------|
| `批量分析A股 {股票1}、{股票2}` | `批量分析A股 药明康德、致尚科技、爱建集团` |
| `A股对比 {股票1} {股票2}` | `A股对比 贵州茅台 五粮液` |

每批次最多 8 只股票，超过自动分两批。

---

### 模式 C：单只美股详细分析

对单只美股进行完整分析，包含基本面、盈利、技术指标（MA50/MA200/RSI）、产业链、关联公司、综合判断、巴菲特框架、估值模型（含 DCF）。

| 触发方式 | 示例 |
|---------|------|
| `分析美股 {ticker}` | `分析美股 AAPL` |
| `美股分析 {ticker}` | `美股分析 NVDA` |
| `{ticker} 怎么样` | `TSLA 怎么样` |

**数据源**：yfinance（财务/行情）+ WebSearch（财报/新闻）+ LLM（产业链）

> 注意：ticker 一律大写（如 AAPL），支持包含字母的美股代码。

---

### 模式 D：多只美股批量对比

| 触发方式 | 示例 |
|---------|------|
| `批量分析美股 {ticker1, ticker2}` | `批量分析美股 AAPL, TSLA, NVDA` |
| `美股对比 {ticker1} {ticker2}` | `美股对比 MSFT GOOGL` |

---

### 模式 E：智能推荐

四种推荐策略，可单独或组合使用。

| 子模式 | 触发词 | 说明 |
|-------|--------|------|
| **E1 超跌筛选** | `超跌推荐`、`跌多了推荐` | 筛选距 52 周高点跌幅 >20% 且基本面未恶化的标的 |
| **E2 新供应商** | `{龙头}新供应商` | 搜索龙头公司（如 NVIDIA/Apple）最新点名的供应商 |
| **E3 大师持仓** | `大师新持仓`、`巴菲特最新持仓` | 搜索知名投资人 13F 最新持仓变动 |
| **E4 大师被套** | `大师被套`、`巴菲特被套` | 检测大师持仓中距高点跌幅 >15% 的标的 |
| **综合推荐** | `推荐个股`、`智能推荐` | 依次执行 E1 → E3 → E4 |

支持指定行业筛选，如 `超跌推荐 半导体`。

---

### 模式 F：牛散持仓分析

分析 A 股知名牛散的持仓动向，按板块汇总共识方向。

| 触发方式 | 示例 |
|---------|------|
| `牛散持仓` | 分析全部 11 位牛散 |
| `{牛散名} 持仓` | `葛卫东 持仓` |
| `牛散持仓 {板块}` | `牛散持仓 半导体` |

牛散名单在 `niushan.yaml` 中自由增删。

> 注意：牛散数据基于季报「十大流通股东」披露，有约 1-2 个月滞后期。

---

### 模式 G：A 股板块分析

对 12 个行业板块进行定量 + 定性分析，输出动量排名和操作建议。

| 触发方式 | 示例 |
|---------|------|
| `分析板块`、`板块排名` | 全板块排名 |
| `哪些板块值得关注`、`板块轮动` | 全板块分析 |

**12 个板块**：锂矿/能源金属、半导体/芯片、AI 算力/光模块、商业航天、消费/食品饮料、医药/生物、新能源/光伏、汽车/自动驾驶、金融/券商、军工/国防、煤炭/能源、稀土/小金属。

板块定义和成分股在 `sectors.yaml` 中自由增删。

---

### 模式 H：三维入场评估（资金 + 技术 + 故事）

针对单只 A 股的入场时机评估，三个维度各有权重（默认资金 35% + 技术 35% + 故事 30%）。

| 触发方式 | 示例 |
|---------|------|
| `值不值得买 {股票}` | `中芯国际 值不值得买` |
| `{股票} 三维分析` | `寒武纪 三维分析` |
| `{股票} 能买吗` | `贵州茅台 能买吗` |

**三维分解**：
- **资金**（35%）：主力净流入趋势、筹码集中度、量价配合
- **技术**（35%）：7 项综合技术判断 + 价托/黄金三角加分 + 年线
- **故事**（30%）：产业链逻辑清晰度、催化强度、供需格局、增量逻辑、业务纯正度

输出包含总分/等级（A≥75/B≥60/C≥45/D<45）、三维共振情况、价托状态、操作建议。

> 仅 A 股可用。美股请用模式 C。

---

### 模式 I：每日涨停复盘 + 概念板块排名

纯数据驱动的每日复盘，无需截图或 LLM。

| 触发方式 | 示例 |
|---------|------|
| `分析涨停`、`涨停复盘`、`今日涨停` | 当日复盘 |
| `概念板块排名` | 概念板块涨跌排名 |

输出四部分：
1. 概念板块涨跌排名（Top 20 涨幅 + Bottom 20 跌幅）
2. 涨停个股全景（按行业分组，含封板时间/连板数/炸板次数/换手率/封板资金）
3. 连板标的（≥2 板）
4. 异常标的（频繁炸板 ≥3 次或换手 ≥20%）

支持板块持续性分析：回溯 10 日热度趋势、5 日存活率。

---

### 模式 J：放量+价托 双条件选股

全市场扫描 A 股（5,500+ 只），筛选同时满足**放量异动** + **近期价托** 的标的。

| 触发方式 | 示例 |
|---------|------|
| `选股`、`筛选`、`放量选股` | 全市场扫描 |
| `创业板选股` | 限定创业板 |
| `选股 2.0倍` | 收紧放量条件 |

**双条件：**
1. 昨日成交量 >= 前日 × 1.8（可自定义）
2. 价托完成（10日上穿20日）<= 10 个交易日内

30 线程并发，全市场约 30 秒完成。结果按放量倍数降序，Top 标的建议跟进模式 H 做三维深度评估。

> 仅 A 股可用。

## 配置文件

| 文件 | 用途 |
|------|------|
| `config.yaml` | LLM API 配置（DeepSeek / 阿里千问） |
| `niushan.yaml` | A 股牛散名单（可增删） |
| `sectors.yaml` | 板块定义及成分股（可增删） |
| `macro_risks.yaml` | 宏观风险因子开关（模式 H 大盘风险过滤） |
| `valuation_reference.yaml` | 行业 PE/PB/EV-EBITDA/WACC 参考值 |
