## 1. EPS — 每股收益

### 是什么？

**公司每年赚的钱，分摊到每一股上是多少。**

把公司想象成一个大蛋糕，EPS 就是切给你的那一小块。

### 怎么算？

```
EPS = 归母净利润 ÷ 总股本
```

**例子：** 茅台2024年净利润约 860 亿，总股本约 12.56 亿股，EPS ≈ 860 ÷ 12.56 ≈ **68.5 元/股**。

### 怎么用？

| EPS | 含义 |
|-----|------|
| > 0 且持续增长 | 公司在赚钱，而且越赚越多 |
| > 0 但下滑 | 赚钱能力在退化 |
| < 0 | 亏损，估值模型基本失效 |

**一句话：** EPS 是所有估值的地基，没有 EPS 就没有 PE、PEG 这些衍生指标。

## 2. PE — 市盈率

### 是什么？

**你愿意为每一块钱的利润，付多少钱。**

这是股票世界最常用的估值指标。PE=10 意味着你以 10 倍的价格买公司一年的利润——**按当前盈利水平，10 年回本**。

### 怎么算？

```
PE = 股价 ÷ EPS
```

### 三种 PE

| 类型 | 分子 | 含义 |
|------|------|------|
| **静态 PE** | 去年年报 EPS | 看过去，有滞后 |
| **TTM PE** | 最近四个季度 EPS 之和 | 看现在，最常用 |
| **动态 PE** | 券商预测的未来 EPS | 看未来，靠猜 |

**例子：** 茅台股价 ¥1650，TTM EPS 约 ¥68.5，PE ≈ 1650 ÷ 68.5 ≈ **24 倍**。

### 怎么用？

| PE 水平 | 常见含义 |
|---------|---------|
| < 10 | 低估值（银行、周期股常态） |
| 10-20 | 合理偏低 |
| 20-30 | 合理偏高（成长股常态） |
| 30-50 | 高估值（需高增长支撑） |
| > 50 | 极高估值（市场给了很高的预期） |

**关键：PE 不能跨行业比！** 银行 PE=6 是正常，芯片 PE=60 也可能是正常。要和**同行业**、**自己的历史**比较。

**常见陷阱：**
- PE 低≠便宜，可能是利润见顶（周期股高点）
- PE 高≠贵，可能是利润低谷（如一次性减值导致 EPS 暴跌）
- 亏损公司 PE 为负或无意义

## 3. PB — 市净率

### 是什么？

**股价相对于公司净资产的倍数。**

净资产 = 总资产 - 总负债，可以理解为"把公司清算了，股东能拿回多少钱"。

### 怎么算？

```
PB = 股价 ÷ BVPS（每股净资产）
```

**例子：** 某银行股价 ¥10，每股净资产 ¥15，PB = 10 ÷ 15 ≈ **0.67 倍**（"破净"——股价低于净资产）。

### 怎么用？

| PB 水平 | 常见含义 |
|---------|---------|
| < 1 | 破净——股价比清算价值还低 |
| 1-3 | 正常范围 |
| 3-8 | 高 PB，常见于品牌消费品（茅台 PB≈8） |
| > 10 | 极高，轻资产公司或市场极度乐观 |

**适用行业：** 银行、保险、地产、钢铁——这些重资产行业用 PB 估值比 PE 更靠谱。

**不适合：** 互联网、软件——核心资产是人/代码/用户，不体现在账面上。

### PB 与 ROE 的关系

高 ROE 的公司通常享有高 PB。逻辑是：能持续创造高回报的资产，市场自然愿意出高价。因此 PB 和 ROE 常常一起看。

## 4. ROE — 净资产收益率

### 是什么？

**公司用股东的钱，一年能赚多少回报。**

这是巴菲特最看重的指标。ROE=20% 意味着股东每投入 100 元，公司一年能赚回 20 元。

### 怎么算？

```
ROE = 净利润 ÷ 净资产 × 100%
```

### 杜邦分析法（拆解 ROE 的来源）

ROE 可以拆成三个因素的乘积：

```
ROE = 净利率 × 资产周转率 × 权益乘数

     赚得多    卖得快     借得多
```

| 驱动因素 | 高说明什么 | 代表行业 |
|---------|-----------|---------|
| 净利率高 | 产品有定价权/品牌溢价 | 茅台、爱马仕 |
| 周转快 | 薄利多销，运营效率高 | 沃尔玛、快消 |
| 杠杆高 | 靠借钱放大收益 | 银行、地产 |

**同一 ROE 可以来自完全不同的商业模式。** 茅台 ROE≈30% 靠的是高净利率；银行 ROE≈12% 靠的是高杠杆。前者显然更安全。

### 怎么用？

| ROE | 评级 |
|-----|------|
| > 20% | 优秀 |
| 15-20% | 良好 |
| 10-15% | 一般 |
| < 10% | 较差 |
| 连续多年 > 15% | 巴菲特选股的重要标准 |

**注意：** 刚上市融资的公司 ROE 会骤降（净资产突然变大），需要等一等再看。

## 5. PEG — 市盈率相对盈利增长比率

### 是什么？

**把你为增长付出的价格也考虑进来的 PE。**

PE=30 贵不贵？如果公司利润每年增长 30%，就不算贵。如果只增长 5%，就很贵。PEG 把"贵不贵"和"长得快不快"放在一起看。

### 怎么算？

```
PEG = PE ÷ 盈利增长率（%）

例如 PE=30，盈利增长率=30%，PEG = 30 ÷ 30 = 1.0
```

### 怎么用？

| PEG | 含义 |
|-----|------|
| < 0.5 | 极度低估（可能是陷阱） |
| 0.5-1.0 | 低估 |
| 1.0 | 合理（Peter Lynch 认为的理想状态） |
| 1.0-2.0 | 偏高 |
| > 2.0 | 高估 |

**关键：** 用**未来 3-5 年的预期增速**而非过去增速。过去的增长不保证未来。

**常见陷阱：**
- 增速用的是预测值，预测可能错
- 周期股利润大起大落，PEG 会失真
- 亏损或微利公司 PEG 不可用

## 6. 自由现金流

### 是什么？

**公司真正能自由支配的现金——赚到的钱减去维持生意必须花的钱。**

净利润≠现金。公司可能账面赚了 100 亿，但要花 80 亿买设备建厂房，真正能分给股东或用来回购股票的只有 20 亿。这 20 亿就是自由现金流。

### 怎么算？

```
自由现金流(FCF) = 经营活动现金流净额 - 资本支出(CAPEX)
```

| 术语 | 大白话 |
|------|--------|
| 经营活动现金流净额 | 做生意真正收到/花出的现金 |
| 资本支出(CAPEX) | 建厂房、买设备等维持/扩张生意的必要支出 |
| 自由现金流 | 剩下的、可以自由支配的钱 |

### 巴菲特视角的"所有者盈余"

巴菲特更进一步提出所有者盈余的概念：

```
所有者盈余 = 净利润 + 折旧摊销 - 维持性资本支出
```

净利润是会计数字，容易调节；自由现金流更真实，更难造假。

### 怎么用？

| 指标 | 含义 |
|------|------|
| FCF > 0 且持续增长 | 健康，公司有真金白银 |
| FCF > 净利润 | 利润质量高（收到的比账面上记的还多） |
| FCF < 0 | 要么在大力投资扩张，要么经营有问题 |
| FCF 长期 < 净利润 | 利润可能"含金量"不足（应收账款多、存货积压） |

**FCF 是 DCF 估值的基础。** 没有正的自由现金流，DCF 模型无法使用。

## 7. EV/EBITDA — 企业价值倍数

### 是什么？

**看整个公司（不只股东）值多少钱，相对于其核心经营利润的倍数。**

PE 只考虑股东权益，EV/EBITDA 把债权人也考虑进来，是更"干净"的估值倍数——不受资本结构（借了多少债）和折旧政策（会计选择）的影响。

### 怎么算？

```
EV = 市值 + 总债务 - 现金
EBITDA = 息税折旧摊销前利润（可以理解为"核心经营利润"）

EV/EBITDA = EV ÷ EBITDA
```

| 术语 | 大白话 |
|------|--------|
| EV（企业价值） | 把整个公司买下来要花多少钱（还债 + 买股票 - 公司账上现金） |
| EBITDA | 不考虑利息、税、折旧、摊销这些"非经营因素"的利润 |

### 为什么不用 PE？

PE 会被以下因素扭曲：
- **资本结构：** 借了很多钱的公司利息支出大→净利润低→PE 虚高
- **折旧政策：** 同样设备，A 公司用 10 年折旧、B 公司用 5 年→利润不同→PE 不同
- **税率：** 享受税收优惠的公司 PE 偏低

EV/EBITDA 排除了这些干扰，更适合**跨公司、跨行业**比较。

### 怎么用？

| EV/EBITDA | 含义（一般情况） |
|-----------|----------------|
| < 8 | 低估 |
| 8-15 | 合理 |
| 15-25 | 偏高（科技/成长股常态） |
| > 25 | 高估 |

**最适合的行业：** 重资产行业——钢铁、化工、航空、电信、能源。

**不适合：** 金融行业（银行/保险，因为债务是经营资产而非融资工具）。

## 8. 股息率

### 是什么？

**你买入股票后，一年能收到多少现金分红，占股价的比例。**

股息率 = 5% 意味着买入 100 元股票，一年能分到 5 元现金。这是"落袋为安"的回报。

### 怎么算？

```
股息率 = 每股分红(DPS) ÷ 股价 × 100%
```

**例子：** 某银行股价 ¥6，每股分红 ¥0.30，股息率 = 0.30 ÷ 6 = **5%**。

### 怎么用？

| 股息率 | 含义 |
|--------|------|
| > 5% | 高股息（银行、煤炭、电力常见） |
| 3-5% | 中等 |
| 1-3% | 偏低（成长股常态） |
| < 1% 或不分红 | 再投资扩张或没钱分 |

### 三个注意事项

1. **看分红可持续性。** 有些公司偶尔一次高分红（卖资产），不可持续。要看**连续 5-10 年的分红记录**。
2. **派息比率** = 分红 ÷ 净利润。> 80% 说明把大部分利润都分了，未来增长靠什么？
3. **除权除息。** 分红后股价会相应下调，不要以为"白拿钱"。

### 股息率在估值中的角色

股息率是 **DDM（股息贴现模型）**的基础：

```
合理股价 = DPS × (1+g) ÷ (r-g)

g = 股息永续增长率，r = 你的要求回报率
```

DDM 适合盈利稳定、分红规律的成熟公司（如公用事业、银行），不适合高成长不分红的公司。

## 9. 毛利率 & 净利率

### 是什么？

**公司每卖出 100 元的产品，能赚多少钱。**

| 指标 | 怎么算 | 含义 |
|------|--------|------|
| **毛利率** | (营收 - 营业成本) ÷ 营收 | 产品本身的赚钱能力 |
| **净利率** | 净利润 ÷ 营收 | 扣完所有费用后最终能留下多少 |

### 例子

茅台一瓶酒卖 1500 元：
- 粮食 + 人工 + 包装 ≈ 120 元（营业成本）→ **毛利率 ≈ 92%**
- 扣掉税、管理费、销售费 → 净利 ≈ 720 元 → **净利率 ≈ 48%**

毛利率 92% 说明产品本身极其暴利；净利率比毛利率低很多，说明税费占比大（白酒消费税高）。

### 怎么用？

**毛利率趋势比绝对值更重要：**

| 趋势 | 含义 |
|------|------|
| 毛利率持续上升 | 产品竞争力增强、涨价、成本控制好 |
| 毛利率持续下降 | 竞争加剧被迫降价、原材料涨价无法传导 |
| 毛利率大幅波动 | 周期行业特征，或经营不稳定 |

**毛利率高（>50%）通常意味着有护城河**——品牌（茅台）、专利（药企）、网络效应（腾讯）、转换成本（微软）。

**净利率看商业模式：**
- 软件/SaaS：30-50%（边际成本极低）
- 消费品品牌：10-25%
- 制造业：5-15%
- 零售/超市：2-5%（靠周转而非利润率赚钱）

## 10. 资产负债率 & 权益乘数

### 是什么？

**公司有多少钱是借来的。**

| 指标 | 公式 | 含义 |
|------|------|------|
| 资产负债率 | 总负债 ÷ 总资产 | 资产中有多少是借的 |
| 权益乘数 | 总资产 ÷ 净资产 | 1 元净资产撬动了多少资产 |
| D/E（美股常用） | 总负债 ÷ 净资产 | 债务是净资产的几倍 |

三者说的是同一件事：**杠杆有多高。**

### 怎么用？

| 资产负债率 | 含义 |
|-----------|------|
| < 30% | 极低杠杆，财务保守 |
| 30-50% | 适中 |
| 50-70% | 偏高（地产、银行常态） |
| > 70% | 高杠杆，需警惕 |

| D/E（美股） | 含义 |
|------------|------|
| < 0.5 | 低杠杆 |
| 0.5-1.5 | 适中 |
| 1.5-3.0 | 偏高 |
| > 3.0 | 危险区 |

**高杠杆是一把双刃剑：**
- 行业好的时候，放大收益（ROE 更高）
- 行业差的时候，放大亏损，现金流断裂风险骤增

**最怕的死亡螺旋：** 利润下降 → 还不起债 → 信用降级 → 融资成本上升 → 利润进一步下降。

**利息保障倍数（补充指标）：** 营业利润 ÷ 利息支出。> 5 比较安全，< 2 需要警惕。

## 11. 指标速查表

### 估值指标

| 指标 | 公式 | 低 | 合理 | 高 | 不适用场景 |
|------|------|-----|------|-----|-----------|
| PE | 股价÷EPS | <10 | 10-25 | >30 | 亏损公司 |
| PB | 股价÷BVPS | <1 | 1-4 | >8 | 轻资产/互联网 |
| PEG | PE÷增速% | <0.8 | 0.8-1.5 | >2 | 亏损/微利/周期股 |
| EV/EBITDA | EV÷EBITDA | <8 | 8-15 | >20 | 金融行业 |
| 股息率 | DPS÷股价 | >5% | 2-5% | <1% | 不分红的成长股 |

### 质量指标

| 指标 | 公式 | 优秀 | 良好 | 警示 |
|------|------|------|------|------|
| ROE | 净利润÷净资产 | >20% | 15-20% | <10% |
| 毛利率 | 毛利÷营收 | >50% | 30-50% | <20%且下滑 |
| 净利率 | 净利÷营收 | >20% | 10-20% | <5% |
| FCF/净利润 | FCF÷净利 | >1 | 0.7-1 | <0.5 |
| 资产负债率 | 负债÷资产 | <30% | 30-50% | >70% |
| D/E | 负债÷净资产 | <0.5 | 0.5-1.5 | >3 |

### 巴菲特选股标准速查

| # | 指标 | 巴菲特标准 | 作用 |
|---|------|-----------|------|
| 1 | ROE | > 15%（连续多年） | 证明有持续竞争优势 |
| 2 | 毛利率 | 稳定或上升 | 证明有定价权/护城河 |
| 3 | 负债 | 低杠杆（D/E < 0.5 更好） | 不需要靠借钱维持 |
| 4 | FCF | 充裕且稳定 | 赚的是真金白银 |
| 5 | PEG | < 1.5 | 为成长付出的价格合理 |
| 6 | 分红 | 持续多年稳定分红加分 | 管理层对股东友好 |
| 7 | 护城河 | 品牌/专利/网络效应/规模 | 阻挡竞争者的壁垒 |

### 同一家公司怎么看？

以茅台为例，假设数据为：股价 ¥1650，EPS ¥68.5，BVPS ¥210，ROE 30%，增速 15%，DPS ¥30。

| 模型 | 计算过程 | 估值结果 | 判断 |
|------|---------|---------|------|
| PE | ¥68.5 × 25倍(白酒行业合理PE) | ¥1712 | 合理 |
| PB | ¥210 × 8倍(茅台历史PB) | ¥1680 | 合理 |
| PEG | PE 24 ÷ 增速 15% = PEG 1.6 | 略高 | PEG>1，增速跟不上估值 |
| 格雷厄姆数 | √(22.5 × 68.5 × 210) | ¥569 | 远高于（品牌溢价） |
| DDM | ¥30×(1+3%)÷(10%-3%) | ¥441 | 远高于（茅台留存利润多） |
| **综合判断** | PE/PB 合理，PEG 偏高 | **合理偏高** | 需增速支撑当前估值 |

**不同估值模型对同一家公司可能给出不同结论——这就是为什么需要多模型交叉验证，而不是只看一个 PE。**
