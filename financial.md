# 极简财务指标

1. 看公司自由现金流/净利润。看是否有很多的应收账款，或者很多的假账。规则：大于1，非常优秀。在0.75—1之间，良好，小于0.75，差。
2. 看扣非净利润/净利润比值，看公司的利润是否都是老老实实做主营业务赚来的。大于0.8优秀。0.5-0.8之间算良好，小雨0.5，算差。
3. 看市净率，股价是否破净，规则：大于1，股价高于净资产。
4. 看净利率，规则：>15% 优秀；5%–15% 一般；<5% 差。

5. 营收同比增长率 Revenue YoY
公式：(本期营收 - 上期营收) / 上期营收
规则：>10% 增长好；0–10% 低速；<0 衰退
6. 净利润同比增长率 Net Profit YoY
公式：(本期净利润 - 上期净利润) / 上期净利润
规则：>10% 优秀；0–10% 一般；<0 变差
